This guide shows the way to write test script based on existing test scenarios with framework
=========================================================================

1. Precondition: install below:
        Pychamr community 2022.3 (or newer),
        python 3.8 or greater

2. down source:

3. Open  test case in source course "\Data\Document\SAULAB_TESTCASE.xlsx"
	- Read testcase ID:  
		+ LG_01 (sheet LOGIN), 
        + HO-02 (sheet HOME)
 
4. Open Test script "\TestScript\Saulab-demo\"
        + test_home_page.py
		+ test_login_page.py

5. Link steps in test case with test script for scripting
6. Run test script:
   At terminal in Pycharm or CMD run:  pytest -s -v .\TestScript\Saulab-demo --alluredir .\Report\

7. Generate report:
   At terminal in Pycharm or CMD run:  allure serve .\Report\
